# MedMate
MedMate is a Web3 solution to manage medical records, consult with professionals, and book appointments with medical experts with different expertise all in one place

## Tech Stack 
1. Solidity
2. IPFS
3. Metamask
4. Ethers.js
5. NFTPort
6. Polybase
7. Typescript
8. NextJS
9. ChakraUI
